# Screenshots para Google Play

## Especificações recomendadas (telefone)

- Mínimo: 2 screenshots
- Formato: PNG ou JPEG 24-bit (sem alpha)
- Resolução recomendada: 1080x1920 (ou proporção similar suportada pela Play)
- Mostrar telas reais do app (sem molduras enganosas)

## Sugestão de capturas

1. Tela inicial / login
2. Perfil do aluno
3. Turmas e presença
4. Mensalidades/pagamentos
5. Loja

## Como capturar (Flutter)

- Emulador Android: `flutter emulators --launch <id>` e usar captura do próprio emulador.
- Dispositivo real: usar captura nativa do Android.
- Garantir idioma PT-BR e dados consistentes para demonstração.

## Tablet (opcional, mas recomendado)

- 7" ou 10" com layout adaptado, se disponível.
